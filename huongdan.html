Ken Mod Z

Các bước chi tiết dành cho Android

  1. Bấm vào liên kết và tải file
  2. Cài đặt ứng dụng ZArchiver
  3. Giải nén file MOD bằng ZArchiver
  4. Giữ vào thư mục đã giải nén và chọn Sao chép → dán vào /Android/Data
  5. Khi ở thư mục /Android/Data, nhấn Dán và áp dụng cho tất cả các tệp để hoàn tất

Các bước chi tiết dành cho IOS

  1. Bấm vào liên kết và tải file
  2. Cài đặt ứng dụng Filza
  3. Giải nén file MOD
  4. Ấn vào file mod cho đến khi thấy file nhân vật ví dụ: optionalab_avatar_66
  5. Giữ vào và ấn sao chép đến file thư mục gốc của game
